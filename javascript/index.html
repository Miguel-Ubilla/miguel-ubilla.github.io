Variables - constantes

Constantes
Una constante se declara utilizando la palabra reservada const, esta reserva un espacio en la memoria para un tipo de dato que permanecera sin cambios durante la ejecucion del programa. 

const nombre = 'Juan Perez';

Estas deben empezar con un guion bajo "_", una letra, o un signo $. La constante se declara solo una vez y puede ser utilizada en cualquier parte del programa si esta es declarada de forma global, no pudiendose actualizar o cambiar su valor.

Variables
Las variables son partes de la memoria reservada para valores que cambiaran conforme se ejecuta el programa, esta se declaran utilizando la palabra reservada let y su declaracion debe cumplir con los mismos parametros de nombre que las constantes. 
let nombre = 'Juan Perez';
//aca se inicializo la variable y en la misma linea se le asigno un valor 'Juan Perez'
// tambien puso realizarse en dos pasos la misma declaracion.
let = nombre;
nombre = 'Juan Perez';

En javascript no es necesario indicar el tipo de dato que almacenaran las variables, esto es reconocido automaticamente por js, los tipos de datos pueden ser:
  • undefined: es una variable o constante declarada pero sin valor definido.
  • null: un valor nulo.
  • booleano: un valor verdadero o falso (true / false).
  • number: un valor numerico que puede ser sujeto a operaciones matematicas.
  • string: una cadena de valores alfanumericos que deben asignarse a la variable entre comillas simples o dobles ('', "").
  • symbol: un tipo de valor que se utiliza para identificar propiedades dentro de los objetos en js.

Cadenas - Strings

Las cadenas o strings como se menciono en el punto de las variables, es una variable o constante cuyo contenido es un valor alfanumerico, el valor se asigna encerrando su contenido entre comillas simples o dobles (''. ""), aunque en ECMAScript6 se introdujo una nueva forma de asignar valores a un string mediante el uso de Backtick (``), que son algo como comillas simples pero en diagonal a la izquierda, la gracia de esta nueva forma es que podemos construir plantillas de strings de una mejor forma que concatenando valores con el operador (+). 

// con el uso de backtick
let nombre = 'Juan Perez';
let saludo = 'buenos dias';
console.log(`al se単or ${nombre}, le deseamos ${saludo}.`)
// Resultado: al se単or Juan Perez, le deseamos buenos dias.

// con el metodo siempre utilizando
// se omite la declaracion de variables ya que son las mismas
console.log('al se単or' + nombre + ', le deseamos' + saludo + '.')
resultado//: al se単or Juan Perez, le deseamos buenos dias.

Como se puede observar la simpleza y limpieza de codigo al utilizar backtick es fenomenal, hay que indicar que las variables se deben incluir encerrando el nombre de la variable entre llaves {} y precedida del signo $, ${variable}
Un aspecto a destacar es que las expresiones matematicas entre variables (si son numericas) que se encuentran entre backtick se ejecutan. 
let numero1 = 2;
let numero2 = 4;
console.log(`el resultado de la suma de ${numero1} + ${numero2} = ${numero 1 +  numero2}`);
// muestra en consola: el resultado de la suma de 2 + 4 = 6
// mostrando el signo + que esta fuera de los ${} solo como un caracter en la cadena, pero 
// ejecutado una operacion matmatica con el signo + que se encuentra dentro de ${}

Tambien es posible insertar codigo html en las plantillas string que utilizan backtick
console.log(`el siguiete texto esta en <strong>negrita</strong>`)

Como ultima caracteristica recomendable tambien esta el uso de cadenas multilinea mediante el uso de backtick solamente, aunque para ser bien representados los saltos de linea en el navegador se debe utilizar una etiqueta <pre></pre>

Condicionales

Son instrucciones que evaluan si una sentencia es verdadera o falsa (true / false) para luego dependiendo de este resultado tomar una u otra accion en la ejecucion del programa.
if es utilizada para analizar variables, operaciones, o propiedades de objetos, su resultado siempre sera un valor booleano (true / false).
Operadores Son parte fundamental en una sentencia If, ya que son los encargados de realizar la comparativa necesaria entre los valores de la sentencia.
Los operadores en Javascript son los siuientes:
= El operador de igualdad asigna un valor a una variable o string. ej: let dia = martes; numero = 1; etc.
== EL operador de igualdad compara dos expresiones o valores que para retornar true deben ser iguales en valor aunque no lo sean en el tipo de dato. ej: numero == "3", esto comrueba que numero sea igual a "3" como cadena, lo que seria lo mismo que numero == 3, comprobado como numero.
===El operador comparativo compara la iagualdad entre expresiones tanto en valor como en el tipo de datos, para que retorne true ambas deben coinidir. ej: numero === "5" retornara false ya que 5 esta entre comillas lo que transforma este en string en vez de numero.
!= Este operador verifica la no igualdad entre expresiones. De no encontrar igualdad retornara True.
> Valida que la expresion de la izquierda sea mayor que la de la derecha.
< Valida que la expresion de la izquierda sea menor que la de la derecha.
>= Valida que la expresion de la izquierda sea mayor o igual a la de la derecha.
<= Valida que la expresion de la izquierda sea menos o igual a la de la derecha. 

if(variableA == variableB) {
// codigo a ejecutar si resultado es true
} else {
// codigo a ejecutar si resultado es false
// puede omitirse este "else" si el programa no debe tomar una accion frente a un resultado false.
}
